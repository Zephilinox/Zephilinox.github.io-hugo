# Zephilinox.github.io-hugo
The hugo files that generate the static webpage

after making any edits:

cd public
git add -A
git commit -m "etc"
git push origin HEAD:master
cd ..
git add *
git commit -m "etc"
git push

when cloning:

cd public
git submodule update
git pull origin master
cd ..
